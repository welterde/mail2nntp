Process stdin (usually an email) to post it on a specified newsgroup server

mail2nntp is available under the GPL (http://www.gnu.org/licenses/gpl.txt).

Available from http://sourceforge.net/projects/mail2nntp/

Please send me an email to report to me the appreciation of this tool and on what kind
of project you use it.

Requires CPAN module News::NNTPClient
DEB package libnews-nntpclient-perl
RPM package NNTPClient
CPAN location http://www.perl.com/CPAN/modules/by-module/News/
